# Troubleshooting

Common blockers:

- Missing `echogalacticsurveyprotocol` artifact.
- Runtime artifact family does not match `standalone`.
- HoloMap layers load without route, fuel, or catalog overlays.
- Manual evidence still contains template timestamps or `TBD` values.
- Launcher logs do not include install, update, repair, and rollback proof.
- Save snapshots do not preserve catalog, route, depot, and salvage state.
