# Runtime Evidence

Public alpha requires real Standalone runtime evidence for survey network loading,
probe launch state, HoloMap overlays, fuel route planning, remote depots,
salvage identification, catalog unlocks, first 30-minute playthrough, first
2-hour playthrough, and Survey Array completion.

The contract smoke in `ECHO-Modules` is necessary but not sufficient. This
edition also needs a visible runtime session with Launcher logs, client logs,
screenshots, and save snapshots.
